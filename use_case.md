# ユースケース記述

## アクター
+ 小売店
+ 受注係
+ 倉庫係
+ 配送係
  + 小売店まで直接届ける

### 受注
+ 受付係が小売店から受注する
  + 小売店の納品希望日（翌日以降）を受け付ける
  + 受注内容を受注票とする
  + 受注票から出荷指示書を作る

### 出荷指示
+ 受付係が出荷指示書を倉庫係に渡す

### 梱包
+ 倉庫係が小売店別に梱包作業をする
+ 倉庫係が小売店宛ての納品書を作る

### 出荷
+ 配送係が倉庫係から小売店宛て納品書を受け取る
+ 配送係が小売店宛てに梱包物を届ける
